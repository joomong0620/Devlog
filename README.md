# Devlog : Developer Community Platform


## 🚀 ZERO-BOOST
> **“0에서 시작해 끝없이 성장하여 1을 만들어내는 팀”**  
> A team that starts from zero, grows endlessly, and creates one.

---

## 🧩 프로젝트 소개

**ZERO-BOOST Devlog**는  
개발자들이 기술을 기록하고, 소통하며,  
성장 과정을 공유할 수 있도록 설계된 **차세대 개발자 커뮤니티 플랫폼**입니다.

기존 개발자 블로그가 기술 전달과 기록 중심이었다면,  
ZERO-BOOST는 **기술 + 감정 + 성장 경험 + 실시간 소통**을 함께 담는 것을 목표로 합니다.

---

## 🧠 기획 배경 및 문제 정의

기존 개발자 플랫폼들은 다음과 같은 한계를 가지고 있습니다.

- 기술 기록 위주의 정형화된 UI/콘텐츠
- 감정, 고민, 성장 스토리를 공유할 공간 부족
- 사용자 참여에 대한 동기부여 구조 부족

개발자는 단순한 코드 생산자가 아니라  
**생각하고, 느끼고, 성장하는 사람**이라는 문제의식에서 출발하여  
ZERO-BOOST는 기술 공유와 함께 **감정·경험·공감의 공간**을 만들고자 기획되었습니다.

## 👩‍💻 My Role (김주연)

### 🧠 핵심 담당 영역

> 💰 커피콩 포인트 결제 시스템
> 📰 지식 뉴스 게시판
> 💼 채용공고 크롤링 시스템
> 🟢 현재 활동중(Online Presence)
> 🤝 소셜 기능 일부 보조



#### 💰 커피콩 결제 & 포인트 시스템 설계

🔹 PortOne 결제 API 연동

- 외부 결제 API 검증 로직 구현
- 결제 요청 → 검증 → 포인트 적립 트랜잭션 처리
- 환불 시 포인트 차감 및 상태 동기화

🔹 백엔드 설계 포인트
- RESTful API 기반 결제 흐름 설계
- 예외 상황에 따른 HTTP 상태 코드 분리
- 트랜잭션 롤백 처리로 데이터 무결성 확보
- 결제/포인트 이력 테이블 분리 설계

👉 단순 API 연결이 아니라 운영 가능한 결제 구조 설계에 집중

#### 📰 지식 뉴스 시스템
- IT 뉴스 자동 수집 (Selenium 기반 크롤링)
- 카테고리 분류 (AI / Tech / IT기업 / 게임 / 과학 등)
- 관리자 승인 기반 게시 구조 설계
- Elasticsearch 연동 검색 최적화

👉 단순 게시판이 아니라 검색 가능한 구조 설계 + 인덱스 설계 경험

#### 💼 채용공고 크롤링 시스템
- Selenium 기반 외부 채용 사이트 데이터 수집
- Oracle DB 적재 자동화
- 중복 데이터 제거 로직 구현
- 일정 주기 자동 업데이트 구조 설계

👉 데이터 파이프라인 설계 경험 

#### 🟢 현재 활동중 (Online Presence)

- WebSocket(STOMP) 기반 실시간 상태 표시
- 로그인 시 세션 등록
- 비정상 종료 시 상태 정리 로직 구현
- 실시간 유저 상태 동기화 처리

👉 실시간 이벤트 기반 구조 설계
